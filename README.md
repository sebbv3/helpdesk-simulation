# Helpdesk Simulations

Collection of real-world IT support scenarios used for learning troubleshooting and diagnostics.

## Scenario 1 - No Internet Connection

Steps:
1. Check physical connection
2. Verify IP address
3. Test gateway connectivity
4. Ping 8.8.8.8
5. Ping google.com
6. Check DNS settings
7. Escalate if required

---

## Scenario 2 - Wi-Fi Not Working

Steps:
1. Verify Wi-Fi is enabled
2. Check airplane mode
3. Forget and reconnect network
4. Verify password
5. Restart adapter
6. Test internet connectivity

---

## Scenario 3 - Printer Not Printing

Steps:
1. Verify power
2. Check cable/network connection
3. Verify printer status
4. Check print queue
5. Restart spooler service
6. Test print

---

## Scenario 4 - Password Reset

Steps:
1. Verify user identity
2. Reset password
3. Unlock account if needed
4. Test login

---

## Scenario 5 - Shared Drive Not Accessible

Steps:
1. Verify network connectivity
2. Check permissions
3. Verify drive mapping
4. Test server access
5. Escalate if required
